## 🚀 Funcionalidades

- 🌀 Sistema de batalha por turnos  
- 🛡️ Execução de *parrys* (defesas perfeitas)  
- ✨ Poder especial dinâmico  
- 🌊 Sistema de *waves* (ondas de inimigos)  
- 🏆 Sistema de pontuação (*score*)
